# 🧠 Learner Adaptation Guide

A lightweight behavioral tool designed to identify **instructional friction** and support adaptive teaching in capacity-building contexts.

This is **not a psychometric instrument** and is not intended for clinical or personality diagnostics.  
It is a rapid instructional support tool to help facilitators adjust delivery methods in real time.

The conceptual framework is informed by established learning and motivational theories, including:

- Kolb (1984) – Experiential Learning
- Freeman et al. (2014) – Active Learning Meta-Analysis
- Ryan & Deci (2000) – Self-Determination Theory
- Vygotsky (1978) – Scaffolding & Social Development
- Cognitive Load Theory

---

## 🎯 Purpose

In many training and capacity-building environments, facilitators must adapt quickly to diverse learning behaviors. However, full psychological profiling is often impractical and unnecessary.

This tool provides a rapid behavioral screener to help answer one practical question:

> How should I adjust my instructional approach so participants can engage effectively?

It focuses on high-leverage observable behaviors that influence how individuals interact with new information, systems, or tasks.

---

## ⚙️ How It Works

1. Participants answer 18 short behavioral questions.
2. The system maps responses to instructional friction patterns.
3. A profile summary is generated with actionable teaching recommendations.
4. Facilitators use the guidance to adjust:
   - Instructional structure
   - Level of autonomy
   - Scaffolding intensity
   - Engagement strategy

The output is designed for immediate application, not archival reporting.

---

## 🚫 What This Tool Is Not

- Not a personality test  
- Not a clinical diagnostic tool  
- Not a validated psychometric assessment  
- Not intended to measure stable psychological traits  

If psychological diagnosis or trait measurement is required, validated psychometric instruments administered by qualified professionals should be used.

---

## 🛠️ Intended Use Cases

- Training & workshop facilitation  
- Capacity-building programs  
- Onboarding sessions  
- Field training environments  
- Adaptive instructional design
